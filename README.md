# doscg-front

### Set up file .env
```
VUE_APP_GOOGLE_KEY=your_key
VUE_APP_API_HOST=https://doscg-kurogeek.herokuapp.com
```

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
